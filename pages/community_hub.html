Optimized QuickSort with 3-Way Partitioning

Just implemented a 3-way partitioning QuickSort that handles duplicate elements more efficiently. Performance improvement of ~40% on datasets with many duplicates! üöÄ

quicksort_3way.py 
def quicksort_3way(arr, low, high):
    if low < high:
        lt, gt = partition_3way(arr, low, high)
        quicksort_3way(arr, low, lt - 1)
        quicksort_3way(arr, gt + 1, high)
