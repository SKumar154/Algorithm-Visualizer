Merge Sort Algorithm

Merge Sort Steps

1
Divide the Array
Split the array into two halves recursively until each subarray has only one element.
2
Merge Subarrays
Combine two sorted subarrays into a single sorted array by comparing elements.
3
Compare and Place
Take the smaller element from either subarray and place it in the merged result.
4
Handle Remaining Elements
Copy any remaining elements from the non-empty subarray.
5
Recursive Completion
Continue merging until all subarrays are combined into one sorted array.

(arr):

arr

def
merge
(left, right):
result = []
i = j = 0

while
i < len(left) and j < len(right):
if
left[i] <= right[j]:
result.append(left[i])
i += 1
else:
result.append(right[j])
j += 1

result.extend(left[i:])
result.extend(right[j:])
return
result
