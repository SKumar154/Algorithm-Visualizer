Binary Search Algorithm

Binary Search Steps

1
Check Array Order
Ensure the array is sorted in ascending order (prerequisite for binary search).
2
Set Boundaries
Initialize left pointer to start (0) and right pointer to end (n-1).
3
Calculate Midpoint
Find the middle element: mid = (left + right) / 2.
4
Compare with Target
Compare the middle element with the target value.
5
Adjust Search Space
If target < mid, search left half; if target > mid, search right half.
6
Repeat or Return
Continue until target is found or search space is empty.
